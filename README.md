## Instalação

```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/IntuiDev-Labs/api-funtransport.git

# Acesse a pasta da aplicação
$ cd funtransport-api
```

```
# Instale as dependências
$ npm install

# Renomei o arquivo .env.example para .env e coloque a sua url de conexão com o banco de dados

# Inicie a aplicação
$ npm run dev

# Ela abrirá em http://localhost:3333

# Utilize o insomnia ou outro programa para testar a api
```
